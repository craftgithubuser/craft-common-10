10.       List and Describe each of the different subsets of SQL(Mean DDL, DML, DCL, TCL)? 
Data definition Language: used to define database 
•	Create: used to form new table or database
•	Alter: used to alter 
•	Truncate: delete data from table 
•	Drop: drops table
•	Rename: renames a table 

Data manipulation language: 
•	Insert : inserts a new row
•	Update: updates existing row
•	Delete: deletes row
•	Select : retrieve record selected. 


Data Control Language: authorization management system 
•	Grant or revoke: used to give or take back autorization

Transaction Control Language: commands used to manage transactions
•	Commit: used to commit action
•	Rollback: used to bring back to specific action.
•	Savepoint: used to save in a specific stage of action
